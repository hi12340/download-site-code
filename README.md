# 网页静态资源下载工具
PowerShell 编写的网页静态资源（CSS/JS/图片/字体等）批量下载工具，仅用于合法合规的技术研究与个人学习。

## 功能说明
- 支持下载远程网页/本地服务器网页的主页面及关联静态资源
- 自动解析 HTML 中的 src/href 资源链接，处理相对路径转绝对路径
- 自动创建资源目录结构，保留原网页资源引用逻辑
- 支持长路径文件名适配（需管理员权限）
- 完善的错误处理与下载统计

## 环境要求
- Windows 系统

## 使用步骤
1. 下载脚本文件
2. 右键以 PowerShell 运行（或在 PowerShell 终端执行）
3. 阅读并确认合规声明（输入 Y 继续）
4. 选择资源保存目录
5. 输入目标 URL（需以 http:///https:// 开头）
6. 确认下载后等待脚本执行完成
7. 资源将保存至选定目录

## 注意事项
1. **合法性前提**：使用前必须确保目标网站资源下载行为符合《网络安全法》《著作权法》等法律法规，不得爬取有版权保护、隐私保护或禁止转载的内容
2. **非商业用途**：严禁用于商业牟利、恶意攻击、批量爬取、流量劫持等违规违法场景
3. **网络限制**：部分网站可能存在反爬机制（如 User-Agent 验证、防盗链、Token 校验），脚本可能无法下载其资源
4. **资源完整性**：动态加载（JS 渲染）的资源无法通过本脚本解析下载，仅支持静态 HTML 中的资源链接
